# 内容结构预览

这份预览用于说明当前站点的“频道 -> 栏目 -> 专题”初始组织方式，方便后续直接接后台管理。

## 1. 当前初始结构

- 频道数：`8`
- 栏目数：`25`
- 专题数：`67`

数据文件：

- [content-structure.json](/Users/k12/Work/Code/k12/nextjs/wangpan/data/content-structure.json)

## 2. 结构层级

采用三层内容树：

- `频道 channel`
- `栏目 category`
- `专题 topic`

说明：

- `频道` 用于首页主导航和频道页
- `栏目` 用于频道下的主要分区
- `专题` 用于承接 SEO 和资源聚合

## 3. 首页推荐展示方式

首页建议展示：

- 顶部搜索框
- 8 个主频道入口
- 精选栏目
- 热门专题
- 最新资源
- 热门标签

其中首页首屏优先展示的主频道建议为：

- 教育考试
- 兴趣技能
- 语言学习
- 语文阅读
- 影视放映
- 软件工具

## 6. 后台管理建议

后续后台建议支持以下能力：

- 新增频道
- 栏目调整顺序
- 专题移动到其他栏目
- 设置 `featured`
- 设置 `status`
- 首页推荐位配置

1. 新增 `channel/category/topic` 表或文件存储

3. 首页和频道页改为从这份结构数据读取
